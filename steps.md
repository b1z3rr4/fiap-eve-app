### Steps

#### Criar uma aplicação react com typescript

```js 
    npx create-react-app my-app --template typescript
```

#### Executar uma aplicação react

```js 
    npm run start
```

#### Criar o código de build de uma aplicação react

```js 
    npm run build
```
